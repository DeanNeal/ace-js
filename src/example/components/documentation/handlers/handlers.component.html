frameworkValue="props.value"
Bind props props.value with certain element
frameworkModel="props.model"
Two way data binding between props.model and html element
frameworkInput="inputData: props.selectedValue"
Allow send props.selectedValue from parent component to child component inputData.
frameworkRef="element"
Add element to ui list. ui - is a list of reference elements
frameworkClass="className: showClass"
frameworkClass="@props.class"
Get 2 params: name of class, boolean(adds class if true). You can use comma(,) symbol to specify several conditions. Also @ is available, so you can bind props to className

Example
frameworkStyle="height: props.height"
Binds props to css rules. Gets 2 params: height - css rule and value - value from props.



frameworkIf="props.isVisible"
Serve to hide or show element; Gets 1 params: value

Element is shown now
frameworFor="props.items"
Repeater. Creates instance for every element in collection props.items

frameworAttr="src: props.image"
Binds attr src with url from props.image
frameworLink="documentation"
Serves for navigation inside application. In this case link leads to documentation page
frameworPattern="[0-9]"
Add error class to the element if value is invalid(doesn't match the pattern)
frameworkOutside="outsideClick"
Trigger methid outsideClick if user clicked beyond the element where frameworkOutside is specified
